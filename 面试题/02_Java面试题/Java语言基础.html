 6、

匿名类有构造函数吗？如果有的话，有几个？


解析：

有；只能有1个；

1）

由于匿名类没有名字，所以无法像声明普通构造函数一样声明。

假如有一个类：

public abstract class Parent {

public abstract void test();

}

我们声明一个继承自Parent类的匿名类的对象：

Parent p1 = new Parent() {

private int age;

//下面的大括号中的就是构造函数

{

System.out.println("Child");

this.age =5;

}

@Override

public void test() {

System.out.println("I am in child class"+this.age);

}

};

p1.test();

上面代码中的：

{

System.out.println("Child");

this.age =5;

}

就是声明构造函数的一种方式，反编译匿名类生成的代码是这样的：

class Main$1

  extends Parent

{

  private int age;

  

  Main$1()

  {

    System.out.println("Child");

    this.age = 5;

  }

  

  public void test()

  {

    System.out.println("I am in child class" + this.age);

  }

}

可以看到编译器帮我们生成了构造函数。

2）假如父类有无参构造函数：

public abstract class Parent {

public Parent(int value)

{

}

public abstract void test();

}


那么匿名的子类创建对象的时候必须给参数赋值：

Parent p1 = new Parent(3) {

private int age;

//下面的大括号中的就是构造函数

{

System.out.println("Child");

this.age =5;

}

@Override

public void test() {

System.out.println("I am in child class"+this.age);

}

};


p1.test();


反编译后是：

class Main$1

  extends Parent

{

  private int age;

  

  Main$1(int $anonymous0)

  {

    super($anonymous0);

    System.out.println("Child");

    this.age = 5;

  }

  

  public void test()

  {

    System.out.println("I am in child class" + this.age);

  }

}

3）从上面的反编译结果可以看出来，匿名类只能有一个构造函数。

 24、

java中，byte类型的取值范围是

A：-256至255
B：-255至255
C：-128至127
D：-127至128
答案：C
解析：

n位二进制数的取值范围是：-2^(n-1)至2^(n-1)-1

其中“x^y”是表示“x的y次幂”

byte是8位二进制，所以是-2^7至2^7-1也就是-128至127


面试的时候一般只会考byte的取值范围，因为short、int等的数太大，不好记。

1）short s1 = 1; s1 = s1 + 1;有错吗?

2）short s1 = 1; s1 += 1;有错吗？ 

解析：

对于short s1 = 1; s1 = s1 + 1;由于1是int类型，因此s1+1运算结果也是int 型，需要强制转换类型才能赋值给short型。因此（1）有错；

说一下值传递和引用传递

解析：

值传递和引用传递都是把变量所表示的值赋值给另一个变量，区别是基本类型变量赋值属于值传递，赋值后值就变成了两份。而对象类型变量赋值时属于引用传递，赋值后两个变量都指向同一个对象
