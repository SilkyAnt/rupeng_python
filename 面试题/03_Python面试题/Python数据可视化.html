谈谈你对数据可视化的理解？


解析：

 数据可视化是指通过可视化表示探索数据，数据可视化主要旨在借助于图形化手段，清晰有效地传达与沟通信息，数据可视化技术的基本思想，是将数据库中每一个数据项作为单个图元元素表示，大量的数据集构成数据图像，同时将数据的各个属性值以多维数据的形式表示，可以从不同的维度观察数据，从而对数据进行更深入的观察和分析。

什么是数据集？数据集常用的两种格式?


解析：

数据集，又称为资料集、数据集合或资料集合，是一种由数据所组成的集合。Data set（或dataset）是一个数据的集合，通常以表格形式出现。数据既可以是数字，也可以是非数字。数据集就是数据的集合，如果是结构化数据就是多行数据的集合，如果是非结构化数据，就是多列数据的集合，一条数据就是组成数据集的最小单元，结构化就是一条数据，非结构化就是一列数据。数据集的集合表现就是一个数据库，一个数据文件。

   数据集常用的两种格式:JSON,CSV
