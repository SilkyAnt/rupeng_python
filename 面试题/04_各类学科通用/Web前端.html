1、

XML 与 HTML 的主要区别

解析：

1. XML是区分大小写字母的，HTML不区分。

2. 在HTML中，如果上下文清楚地显示出段落或者列表键在何处结尾，那么你可以省略</p>或者</li>之类的结束 标记。在XML中，绝对不能省略掉结束标记。 

HTML：<img src="1.jpg"><br><br>

XML：<img src="1.jpg"></img><br/><br/>

3. 在XML中，拥有单个标记而没有匹配的结束标记的元素必须用一个 / 字符作为结尾。

4. 在XML中，属性值必须放到在引号中 <Person name="rupeng">。在HTML中，引号是可用可不用的<input name=rupeng>。

5. 在HTML中，可以拥有不带值的属性名<input type="checkbox" checked/>。在XML中，所有的属性都必须带有相应的值。<Person checked="true"/>

 

如果使用了完全符合XML语法要求的HTML，那么就叫做符合XHTML标准。

关于function的问题

（1）

var x=1;

var y=0;

var z=0;

function add(n){n=n+1;return n;}

y=add(x);

function add(n){n=n+3;return n;}

z=add(x);

求y和z的值是多少？

2）

var x=1;

var y=0;

var z=0;

var add = function(n){n=n+1;return n;}

y=add(x);

add =function(n){n=n+3;return n;}

z=add(x);

求y和z的值是多少？


解析：

1）JavaScript引擎会先解析所有的命名函数，再去从上向下执行js代码。所以第二个add覆盖了第一个add的定义。因此结果是4、4

2）这个和上面不一样，这个是匿名函数，add指向谁,add(x)就执行谁，所以结果是2、4

命名函数的add就是函数的名字。而var add = function(n){n=n+1;return n;}则是创建一个匿名函数，并且把变量add指向它，这个函数还是匿名函数，并不会因为有一个变量指向它他就不是匿名函数了。

 5、

HTML 的 form 提交之前如何验证数值文本框的内容不空且全部为数字? 否则的话提示用户并终止提交

解析：

方法一，使用正则表达式

/^\d+$/.test(value) 

因为test是部分匹配，即value中只需要有一部分和正则匹配就会返回true，所以需要加上^和$保证完全匹配

 

 

方法二：

<script type="text/javascript">

    document.getElementById("form01").onsubmit=function(){

        var value = document.getElementById("phone").value;

        //如果value为空

        if(!value){

            alert("手机号不能为空");

            return false;

        }

        for(var i=0;i<value.length;i++){

            var ch = value.charAt(i);

            //字符串直接比较时是按字典顺序进行比较的

            if(ch<"0" || ch>"9"){

                alert("只能输入数字");

                return false;

            }

        }

    }

</script>

 8、

手动写一个函数用来发送ajax请求（不用jQuery）

解析：

function mypost(url,param,successFunc){


var xmlHttp ;


if(window.XMLHttpRequest){

xmlHttp   = new XMLHttpRequest();

}else{

xmlHttp = new ActiveXObject("Microsoft.XMLHTTP");

}


xmlHttp.onreadystatechange=function(){

if(xmlHttp.readyState==4 && xmlHttp.status==200){

successFunc(xmlHttp.responseText);

}

}


xmlHttp.open("POST",url,true);

xmlHttp.send(param);

}
