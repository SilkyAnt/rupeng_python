网络通讯中tcp udp的区别是什么？

解析：

1、TCP是需要先建立连接的;UDP是无连接的，即发送数据之前不需要建立连接

2、TCP提供可靠的服务。也就是说，通过TCP连接传送的数据，无差错，不丢失，不重复，且按序到达；UDP不保证数据的传输的完整性；

3、TCP效率低，UDP效率高；

 7、

有下面的代码：

int k=0;

int ret = ++k+k++ + ++k+k;

执行完后ret的值为多少

解析：

答案：8

++运算符优先级高于+，因此用()明确一下优先级为：

int ret = (++k)+(k++) + (++k)+k;

++k表达式的值为自增之后的值，k++表达式的值为自增之前的值；表达式从左向右执行；

咱们从左向右分析：

(++k)执行后表达式的值为1，执行后k=1

(k++)执行后表达式的值还是1，执行后k=2

(++k)执行后表达式的值为3，执行后k=3

所以int ret=1+1+3+3=8


这样的代码只会考试的时候考，工作中写这样的代码会被打死。

 12、

有以下方法的定义，请选择改方法的返回类型

ReturnType method(byte x,double y)

{

   return (short)x/y*2;

}


A：byte
B：short
C：int
D：double
答案：D
解析：

()用来进行显式转换是修饰离他最近的变量，因此(short)x/y*2等价于((short)x)/y*2


((short)x)是short类型、y是double类型，2是int类型，short、double、int进行运算，结果是最大的精度、最大的长度，因此结果是double类型


 13、

int i=12;

int j=(i*=++i);

请问j的值是多少？

解析：

答：156

解析：

int j=(i*=++i);

等价于：int j=(i=i*(++i));

表达式是从左向右：所以是int j=(i=12*13);因此i=12*13之后i=156，“赋值表达式也有值，其值是赋值之后左边的值”，所以j=156
